Making Webpages Interactive (DOM)

Webpage design was cranked up a notch in this course where the script tag for html was discussed. Techniques such as selecting elements by different attributes and event listeners now allowed webpages to be more than just stationary images. Libraries were also covered, allowing you to re-use someone else's code.

Making Webpages Interactive with jQuery

Similar to the DOM course on Khan Academy, this course explored jQuery. a popular javascript library used in webpage design.

Algorithms

This more theoretical Khan Academy course looked into algorithms, what they are and how to improve efficiency. Various sorting algorithms such as Selection Sort and Quick Sort were taught and implemented along with the introduction of graph theory and asymptotic notation.

Intro to SQL

Databases are of utmost importance in today's world and one of the most effective ways to create and access specific data from a database is SQL. In this course, database creation along with various forms of querying were introduced.

Intro to JS: Free Code Camp

The majority of ICS4U had been done on Khan Academy and while it is a great resource, changing it up once in a while helps break up the monotony of Khan Academy's teaching format. Free Code Camp's structure focuses more on giving you the raw tools and then allowing you to problem solve with them in the projects. This was different from Khan Academy's approach of trying to guide you to an exact destination. This intro to JS course gave more basic functionalities of Javascript including string manipulation and case statements. It was a nice breath of fresh air.
